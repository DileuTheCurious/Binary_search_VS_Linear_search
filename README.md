# Binary_search_VS_Linear_search
Add binary and linear search project in Python

This project compares two search algorithms in Python:

- Binary Search
- Linear Search

## Features

- Searches for a user-defined number
- Returns the position of the number in the list
- Measures execution time in milliseconds
- Compares binary search performance with linear search

## Technologies Used

- Python
- time module
- Log(O)n (Binary search)
- (O)n (Linear search) 

## Example

- Test 1: searches for `500000`
- Test 2: searches for `1000001`
- User can input any number from 1 to 1000000

## Goal

The goal of this project is to demonstrate how binary search is much faster than linear search when working with large ordered lists.
